# Everyware

 Everyware 는 일반 그룹웨어의 틀에 갇히지 않고, 자유롭게 회사 환경에 맞
게커스텀이 가능하며, 기능과 기능이 서로 연결되어 더 낳은 업무 환경을제공하고, 
각 기능 마다 관리자 페이지를 두어, 사후 관리까지 제공하는 그룹웨어 시스템입니다.


기술스택: Spring, Spring- security, spring-websocket, mybatis, Bootstrap, jQuery
